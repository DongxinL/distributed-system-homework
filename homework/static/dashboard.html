实验说明

推荐按以下顺序验证第六次作业功能。

1. 点击“刷新治理配置”，确认 Nacos 下发的动态参数已显示。
2. 访问“验证动态路由”，通过服务名路径检查 Gateway Discovery Locator 是否生效。
3. 连续点击“热点限流测试”，观察返回结果与 429 限流行为。
4. 点击“熔断降级测试”，Gateway 会对超时接口做降级兜底。
